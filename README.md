## Описание проекта

Данный проект создан в рамках лабораторных работ по изучению CSS-фреймворка Bootstrap 5

## Что было изучено

- Подключение Bootstrap через CDN
- Система сеток (Grid System)
- Компоненты: Navbar, Cards, Forms, Modal, Accordion
- Утилитные классы
- Адаптивные изображения
- Валидация форм
- Кастомизация стилей

## Структура проекта

```
isrpo_lab12_13/
├── index.html        # Главная страница
├── experiment.html   # Эксперимент с карточкой товара
├── gallery.html      # Галерея с фильтрацией
├── timeline.html     # Временная шкала достижений
├── portfolio.html    # Финальное портфолио
├── custom.css        # Кастомные стили
├── README.md         # Описание проекта
└── img/              # Скриншоты
```

## Как запустить

Откройте любой HTML файл в браузере или используйте Live Server в VS Code

## Технологии

- HTML5
- CSS3
- Bootstrap 5.3.8
- Bootstrap Icons 1.11.3
- JavaScript (для валидации форм и галереи)
